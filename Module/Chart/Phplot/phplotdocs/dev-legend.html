Chapter 8. PHPlot Legend Layout

This figure illustrates the layout of the legend, as drawn by DrawLegend(). Two of six possible arrangements of the colorboxes and text are shown. See SetLegendStyle for more information.

Through PHPlot-5.2.0, the width of the colorboxes was always char_w, the width of one character (or the width of the upper case letter "E") in the legend text font. Starting with PHPlot-5.3.0, this can be adjusted using a class variable. See Section 4.5.3, “Tuning the Legend” for more.
