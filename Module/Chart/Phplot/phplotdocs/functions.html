Chapter 6. PHPlot Functions By Category

This chapter presents a grouping of the PHPlot functions by category. Descriptions of the functions can be found in PHPlot Function Reference.

6.1. Core

The functions in this section form the core of PHPlot. You cannot produce plots without using these. If you want to produce plots with little or no customization, you need use no other functions. These functions are used to create a PHPlot object, select the type of plot you want, provide the data to be plotted, and output the plot. For more information, see Section 3.2, “Programming Overview”.

  • DrawGraph
