6.11. Labels

The functions in this section control the formatting and display of labels. This includes data labels (axis data labels which come from your data array, and data value labels), tick labels (automatically generated by PHPlot), and the special data labels used on pie charts to show the percentage of each pie slice. For more information, see Section 3.6, “Labels”.
